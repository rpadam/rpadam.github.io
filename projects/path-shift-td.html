Path Shift TD

A standalone tower defense game where lanes rotate every few waves and your tower combos matter. Build to survive wave pressure, then trigger overdrive to recover from difficult spikes.

The game is versioned and deployed from its own repository.

Launch Game

Core Mechanics

  • Three tower classes with distinct roles
  • Path shifts every third wave
  • Overdrive meter and timed burst windows
  • Risk crates for economy boosts
